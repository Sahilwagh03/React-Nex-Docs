## Props

### `position`

- Type: `string`
- Description: Determines the position of the tooltip relative to the trigger element. Accepted values are `top`, `bottom`, `left`, and `right`. Default is `top`.

### `className`

- Type: `string`
- Description: Additional Tailwind CSS classes to apply to the tooltip container.

### `onClick`

- Type: `function`
- Description: Callback function to be executed when the component is clicked.
