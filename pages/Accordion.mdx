## Props

### `value`

- Type: `string`
- Description: Unique identifier for each accordion item.

### `type`

- Type: `'nested' | 'single'`
- Description: Specifies the type of accordion layout. If set to `'nested'`, it allows nesting of accordion items. Defaults to `'single'`.
