## Props

### `enabled`

- Type: `boolean`
- Description: Controls whether the switch is enabled or disabled.

### `onChange`

- Type: `(state: boolean) => void`
- Description: Callback function that is called when the switch is toggled. The new state is passed as a boolean.

### `disabled`

- Type: `boolean`
- Description: If true, the switch is not clickable and appears in a disabled state.
