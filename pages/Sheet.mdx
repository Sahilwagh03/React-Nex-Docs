## Props

### `side`

- Type: `'top' | 'bottom' | 'left' | 'right'`
- Description: Specifies from which side the sheet slides in. Defaults to `'right'`.

### `children`

- Type: `React.ReactNode`
- Description: The content to be displayed inside the sheet.
